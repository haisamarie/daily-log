## 型の基礎

### type（型エイリアス）

#### どういうもの?

型に名前をつけて再利用可能にする機能

#### いつ使うの？

- プリミティブ型やユニオン型に名前をつけたいとき
- 関数の型を定義したいとき
- タプルやマップ型を定義したいとき

```ts
// プリミティブ型のエイリアス
type UserID = string;
type Age = number;

// オブジェクト型
type User = {
  id: UserID;
  name: string;
  age: Age;
};

// ユニオン型
type Status = 'pending' | 'approved' | 'rejected';

// 関数型
type Formatter = (value: string) => string;

// タプル
type Point = [number, number];
```
